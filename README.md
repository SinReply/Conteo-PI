# Conteo-PI

Pon a prueba tu memoria de los dígitos de PI con este juego de consola.  
Incluye tanto la versión en [Python](./PI_Main.py) como un [ejecutable para Windows](./PI_Main.exe).

---

## Archivos principales

- [`PI_Main.py`](https://github.com/SinReply/Conteo-PI/releases/download/v.1/PI_Main.py) → Código fuente en Python.
- [`PI_Main.exe`](https://github.com/SinReply/Conteo-PI/releases/download/v.1/PI_Main.exe) → Versión ejecutable para Windows.

---

# Cómo usar

## Con Python:

1. Asegúrate de tener Python instalado.
2. Ejecutalo en la terminal, CMD ó usa algún IDE o editor de código funcional:

```
python PI_Main.py

```

## Con Windows (.exe):

- Haz doble clic en [`PI_Main.exe`](https://github.com/SinReply/Conteo-PI/releases/download/v.1/PI_Main.exe) o ejecútalo desde la terminal de Windows y sigue las instrucciones.

---

## Cómo funciona

- El programa pide al usuario que ingrese los primeros 54 dígitos de PI.
- Por cada dígito ingresado:
  - ◯ → dígito correcto
  - ✕ → dígito incorrecto (el programa indica la posición del error)
- Si todos los dígitos son correctos, muestra el mensaje **"Zero Errores"**.
- Puedes cerrar el programa tras revisar tu resultado.
